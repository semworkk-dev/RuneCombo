# РунаКомбо - Славянская Магическая Игра

Полноценное Android-приложение в стиле славянского фэнтези с механикой match-3, героями, рунами и эпическими боссами.

## 🎮 Основные Возможности

### Игровой Процесс
- **Match-3 Механика**: Соединяй магические руны по 3 и более для создания комбо
- **Комбо-Система**: Множители урона за большие комбинации
- **6 Типов Рун**: Огонь, Вода, Природа, Молния, Свет, Тень

### Герои
- **6 Уникальных Героев**: Богатырь, Домовой, Лиса, Алиса, Колобок, Кот Баюн
- **Система Способностей**: Каждый герой имеет уникальную способность
- **Прокачка**: Повышение уровня и звезд редкости

### Контент
- **8 Локаций**: От Деревни Домового до Башни Кощея
- **100+ Уровней**: Прогрессивная сложность
- **Боссы**: Эпические сражения каждые 10 уровней
- **Коллекции**: Собирай героев и руны

### Технические Особенности
- **Оффлайн-Режим**: Играй без интернета
- **Синхронизация с Сервером**: Сохранение прогресса в облаке
- **Оптимизация**: Размер приложения < 100 МБ
- **Высокая Графика**: Jetpack Compose с анимациями

## 🛠️ Технический Стек

## 🚀 Установка и Запуск

### Требования
- Android Studio 2023.1+
- JDK 17+
- Android SDK 34

### Шаги Установки

1. **Клонируй репозиторий**
```bash
git clone <repository-url>
cd RunaCombo
```

2. **Откройте в Android Studio**
- File → Open → выберите папку RunaCombo

3. **Синхронизируй Gradle**
- Build → Rebuild Project

4. **Запустите приложение**
- Run → Run 'app'
- Выберите эмулятор или подключенное устройство

## 🎨 Кастомизация

### Изменение API URL
В `di/AppModule.kt` измените:
```kotlin
.baseUrl("https://api.runacombo.com/")
```

### Изменение Цветов
В `ui/theme/Color.kt` измените палитру цветов

### Добавление Новых Героев
В `domain/model/Hero.kt` добавьте нового героя в `HeroPresets`
